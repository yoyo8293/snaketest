## 6. 常见问题

### Q: OpenCode 中 `&&` 语法报错？

**原因：** PowerShell 不支持 `&&` 语法

**解决：** 使用 `;` 或分两行执行

```powershell
# 错误
cd snaketest && opencode

# 正确（PowerShell）
cd snaketest; opencode

# 正确（Bash）
cd snaketest && opencode
```

### Q: gh 命令找不到？

**原因：** winget 安装后需要使用完整路径或刷新 PATH

**解决：** 使用完整路径执行

```bash
"C:\Program Files\GitHub CLI\gh.exe" --version
```

### Q: Git 换行符警告？

**警告：** `LF will be replaced by CRLF`

**解决：** 这是 Windows 正常行为，不影响功能，可忽略
